Select any API below to explore and view live data from the FilmRental backend.

Customer Rentals
View rental records for a specific customer by ID.
GET /api/v1/customers/{id}/rentals
🎭
Film & Actors
Film details with all associated actors by film ID.
GET /api/v1/films/{id}/actors
💳
Payment Details
Payment records with customer info and amounts.
GET /api/v1/payments/details
🧑‍💼
Rental Staff
Rental records with the staff member who handled each.
GET /api/v1/rentals/staff
🎞️
Rental Films
Rental records linked to film titles and return dates.
GET /api/v1/rentals/films
🔗
Backend API is running on http://localhost:8081  ·  All requests are authenticated via JWT Bearer Token.
